Facebook page plugin

== Description ==

Facebook page plugin is available to WordPress shortcode and widget.

[Developers Facebook - Page Plugin](https://developers.facebook.com/docs/plugins/page-plugin)

= Available shortcodes: =

`[lana_fb_page href="https://facebook.com/lana.codes/" width="" height="" tabs="" hide_cover="false" show_facepile="false" small_header="false" adapt_container_width="true"]` Facebook page


= Lana Codes =
[Lana Facebook Page](http://lana.codes/lana-product/lana-facebook-page/)

== Installation ==

= Requires =
* WordPress at least 4.0
* PHP at least 5.3

= Instalation steps =

1. Upload the plugin files to the `/wp-content/plugins/lana-facebook-page` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

= How to use it =
* in `Appearance > Widgets`, add the 'Lana - Facebook Page' widget to the sidebar, for example, add it to the Primary Sidebar.
